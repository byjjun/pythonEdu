4. 여러곳에서 써먹으니 github 연결하려니 근데 에러 발생!!
eclipse : cannot open git-upload-pack
window > preference > team > git > configuration
add entry의 key, value에 다음과 같이 넣자.
 - key : http.sslVerify
 - value : false
